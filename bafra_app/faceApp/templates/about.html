About B A F R A



The image of a face in a passport is the only data to use for personal authentication worldwide, regardless of nationality. In addition, image recognition on the face offers a unique advantage over other personal authentication technologies because it can authenticate people at a distance. B A F R A Application presents facial recognition technology in real time to use in a point system.

The creation of B A F R A

The choice of programming language fell on Python. For the realization of our project. This application was created thanks to very professional modules, facial recognition which uses cnn technology (convolutional neural network) with GPU support to give us a very good result.

The library uses

For the realization of our project, several libraries were forced to use it: OpenCV, Dlib and face_recognition. Dlib is a library that can be used for free in multiple applications. It contains machine learning algorithms and tools that help us create complex software to solve real world problems. OpenCV (for Open Computer Vision) is a free, open source graphics library, originally developed by Intel, specializing in real-time image processing. The library, created by Adam Geitgey, wraps around the facial recognition feature of dlib, making it easier to work with.
